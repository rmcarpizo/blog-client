# 📝 Article Blog App

A full-stack blog application built with the **MEVN Stack**. Users can register, login, create posts, and comment. Admins can manage all content via a dashboard.

---

## 🚀 Live Demo

- **Frontend:** https://blog-client-six-alpha.vercel.app

---

## ✨ Features

- User registration and login (JWT authentication)
- Create, edit, and delete blog posts
- Comment on posts
- Admin dashboard to manage all posts and comments
- Real-time form validation

---

## 🛠️ Tech Stack

- **Frontend:** Vue 3, Vue Router, Pinia, Axios, Bootstrap 5
- **Backend:** Node.js, Express.js, MongoDB, Mongoose, JWT

---

## ⚙️ Setup

```bash
# Server
cd server
npm install
cp .env.example .env  # fill in your values
npm run dev

# Client
cd client
npm install
npm run dev
```

---

## 🌐 Deployment

- **Frontend** → Vercel
- **Backend** → Render
- **Database** → MongoDB Atlas
